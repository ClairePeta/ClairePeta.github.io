RMIT Library Tour

I was offered contract work from Prof. Zambetta in a new team he was starting up; Research & Innovation in Games (RIG Lab). I worked as a contract junior programmer, on the team working closely with the experience team leaders, learning from them as I got hands on implementation experience. This project also used voice commands, hand gestures and Vuforia for object tracking, the project involved scanning the two levels of the RMIT and arranging objects in Unity so that when the project was running on the Hololens it would align with the room. This project is the second base environment of the Mixed Reality course at RMIT.

  • My Contributions: Environment Scanning, Hologram Alignment, Trigger Areas, Testing
